//进入 usr文件夹
psftp> cd /usr
Remote directory is now /usr

//创建test文件夹
psftp> mkdir test
mkdir /usr/test: OK

//上传d:\desktop\test.html（本机路径） 到/usr/test/test.html（目标路径）
psftp> put d:\desktop\test.html /usr/test/test.html
local:d:\desktop\test.html => remote:/usr/test/test.html
至此，文件上传完成。
